# 十六、高频 Bug 排查清单（20类）

适用于快速定位问题：

- 页面数据不刷新
- 表单回显失败
- watch 不触发
- computed 不更新
- props 修改不生效
- this 指向错误
- 接口成功但列表未更新
- 弹窗关闭后数据残留
- v-if / v-show 异常
- 生命周期执行异常
- Vuex 数据不同步
- 分页组件不生效
- selection 多选异常
- 动态表单校验失效
- 路由参数丢失
- keep-alive 异常
- 列表重复渲染
- 接口重复请求
- axios 重复发送
- 页面刷新状态丢失

建议配合 Bug 排查模板使用。
