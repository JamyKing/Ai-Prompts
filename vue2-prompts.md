- Vue2

# 一、万能总模板（推荐置顶 ⭐）

```text
请作为一名有企业后台项目经验的前端工程师，
基于现有 Vue2 代码帮我完成需求。

要求：
1. 保持 Vue2 options API 写法
2. 尽量最小改动
3. 保持现有项目风格
4. 给出完整可运行代码
5. 标出修改位置
6. 说明风险点和测试点

代码如下：
【代码】

需求如下：
【需求】
```
